# Images Directory Structure

This folder contains all organized images for the bundle builder application.

## Folder Structure

- **products/**: Main product images (high resolution)
- **variants/**: Product variant/color option images
- **icons/**: UI icons and badges

## Image Guidelines

- Use WebP format for best performance
- Product images: 800x800px (1:1 aspect ratio)
- Variant images: 100x100px (1:1 aspect ratio)
- Icons: SVG format preferred
